Azure Bastion is a fully managed Platform-as-a-Service (PaaS) offering that provides secure and seamless RDP and SSH connectivity to Azure virtual machines over TLS directly from the Azure portal, eliminating the need for public IP exposure and reducing the attack surface.

This repository contains useful templates, artifacts and queries to improve the Azure Bastion experience.
